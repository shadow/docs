Module vasi_sync::sync

source ·
Expand description

This is public primarily for the integration tests in tests/*, which is the recommended way of writing loom tests. Not actually intended for usage by other crates. Synchronization primitives that are modeled in loom

This module provides some very low-level primitives, such as atomics, and futex. When testing under loom they model the corresponding operation in loom instead of executing it natively.
