A parser for the Graph Modelling Language (GML) format.

Example graph:

graph [
  directed 1
  node [
    id 0
    label "Node 0"
  ]
  node [
    id 1
    label "Node 1"
  ]
  edge [
    source 0
    target 0
  ]
  edge [
    source 1
    target 0
  ]
]
