pub unsafe extern "C" fn childpidwatcher_watch(
    watcher: *const ChildPidWatcher,

    data: *mut c_void
) -> WatchHandle
Expand description

Call callback exactly once from another thread after the child pid has exited, including if it has already exited. Does not reap the child itself.

The returned handle is guaranteed to be non-zero.

Panics if pid doesn’t exist.
