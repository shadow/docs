Module shadow_rs::utility::byte_queue[−][src]

Expand description

A shared buffer that is composed of several chunks. The buffer can be read and written and guarantees it will not allow reading more than was written. Its basically a linked queue that is written (and grows) at the front and read (and shrinks) from the back. As data is written, new chunks are created automatically. As data is read, old chunks are freed automatically.
