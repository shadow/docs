Values for working with a simulated duration. Use EmulatedTime to represent an instant in time.

In Rust, use EmulatedTime to represent an instant in time, or SimulationTime to represent a time interval. SimulationTime is meant to replace c::SimulationTime from the C APIs.

This module contains some identically-named constants defined as C macros in main/core/support/definitions.h.
