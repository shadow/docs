source

pub fn for_each<F>(&mut self, f: F)
where F: FnMut(HostType) -> HostType,

For each Host, calls f with the host. The Host must be returned by the closure. The ownership of the Host is transferred in and out of the closure rather than using a mutable reference since Shadow needs to put the host in a global with 'static lifetime (the worker).
